# BepInEx.DiscordSocialSDK
Wrapper around [Discord Social SDK](https://discord.com/developers/social-sdk) for [BepInEx5](https://github.com/BepInEx/BepInEx)

# How to install
1) Install zip from [releases page](https://github.com/Rostmoment/BepInEx.DiscordSocialSDK/releases)<br>
2) Install [BepInEx5](https://github.com/BepInEx/BepInEx)<br>
3) Drag BepInex folder to game folder<br>
4) Open game and [connect your accound to application](https://i.ibb.co/DHwZRJ7r/image.png)

# How to use in code
This library includes every single method from original Discord Social SDK, but with some wrappers for extra convenient<br>
Add it to dependencies and use it!<br>
All lwrappers are described in `.xml` file or in [Wiki](https://github.com/Rostmoment/BepInEx.DiscordSocialSDK/wiki)
Everything else you can find imn [Discord's documentation](https://discord.com/developers/docs/social-sdk/index.html)
